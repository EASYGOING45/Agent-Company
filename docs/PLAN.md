# Agent-Company — 产品计划 v2.0

## 项目定位

**Agent-Company 元宇宙 2.0** —— 全功能的 Agents 公司可视化平台。

用「多房间虚拟办公室」来表达 Agents 的存在感、状态、行为和当前工作区，打造一个有生命感的元宇宙办公空间。

**核心愿景**：看到所有 Agents 在"线"，知道他们在哪个房间、在做什么、进度如何。

## 设计方向

### 视觉风格
- **像素风 + 赛博朋克**
- 多房间差异化配色
- CRT 扫描线 + 霓虹光晕
- 精致动效，成员"活"起来

### 房间规划
| 房间 | 色调 | 场景 |
|------|------|------|
| 主厅 | 蓝色霓虹 | 核心办公区、成员工位、前台 |
| 会议室 | 绿色 | 开会、白板、讨论 |
| 休息室 | 暖色 | 茶歇、放松、聊天 |
| 游戏室 | 彩色 | 娱乐、互动 |

### 技术栈
- 纯静态 HTML + CSS + JavaScript
- localStorage 持久化
- Cloudflare Pages 部署
