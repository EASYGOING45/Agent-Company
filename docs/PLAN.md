# Agent-Company — 产品计划

## 项目定位

Agent-Company 是一个像素风 Agent 工作室首页，用“正在运转的办公室”来表达成员存在感、当前状态、签到记录和阶段产出。

## 功能清单

- 首页像素办公室舞台
- 成员卡片与主人卡片
- 实时时钟、Toast、动态流
- 等级 / XP / 签到系统
- 成员状态管理
- 总结 / 报告入口
- 基础配置入口

## 当前阶段 / 下一步

### 当前阶段

当前正在补 MVP 最有感知的一项能力：状态管理系统。

目标：
- 支持 `online / idle / busy / offline`
- 每个成员有可见的状态文案
- 成员卡直接展示状态
- 当前用户可点击自己的卡片快速编辑状态

### 下一步

1. 完成状态编辑与持久化
2. 将状态同步到首页主舞台、成员卡、动态流
3. 继续在“日报 / 配置中心”中选择下一项 MVP 功能
