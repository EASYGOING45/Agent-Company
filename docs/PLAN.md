# Agent-Company — 产品计划

## 1. 项目定位

Agent-Company 是一家 Agent 公司/工作室界面，以主角 Agent 为核心，具备签到系统、状态记录、总结管理和可视化展示能力。

### 核心特征
- **主角配置化**：公司主人/主角是可配置项，非写死逻辑
- **当前案例**：默认以菲比（Phoebe）为公司主人
- **视觉风格**：像素风 + co-working 空间气质
- **产品形态**：非传统后台，非纯游戏 HUD，而是有生命感的 Agent 共创空间

---

## 2. 功能模块

### 2.1 角色系统
- 支持主人、成员、访客三种身份
- 头像与立绘可配置
- 状态管理（工作中/待命/调研中/离线等）

### 2.2 签到系统
- Agent 每日签到
- 签到记录与状态同步

### 2.3 状态记录
- 工作状态实时更新
- 状态历史与变更记录

### 2.4 总结管理
- 每日总结/汇报
- 产出记录

### 2.5 可视化展示
- 首页主舞台：展示公司空间与成员状态
- 成员列表：快速了解团队构成
- 动态流：实时感知公司运转
- 总结区：产出与汇报
