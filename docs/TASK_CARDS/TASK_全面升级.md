## 🎯 任务目标

将 Agent-Company 升级为**全功能的 Agents 元宇宙可视化平台**，核心能力：

1. **多房间系统** - 虚拟办公室多个房间（主厅/会议室/休息室/游戏室）
2. **实时状态可视化** - Agents 的状态、行为、当前工作区
3. **增强交互** - 房间切换、成员互动、事件系统
4. **UI 全面升级** - 更精致的像素风 + 更好的动效

## 📦 涉及文件

- `src/data.js` - 房间数据模型、成员状态扩展
- `src/main.js` - 房间切换逻辑、状态更新、事件系统
- `src/styles.css` - 房间样式、动效、响应式布局
- `index.html` - 多房间入口、导航、房间场景
- `public/assets/` - 新增房间背景/装饰资源

## 📋 执行步骤

### Phase 1：房间系统基础

1. **data.js** - 添加房间数据模型：
   - 房间列表（主厅、会议室、休息室、游戏室）
   - 每个房间的场景描述、成员列表
   - 房间切换状态

2. **styles.css** - 添加房间样式：
   - 每个房间独特的背景/配色
   - 房间过渡动画
   - 像素风房间元素（桌子、窗户、装饰）

3. **index.html** - 添加房间导航：
   - 顶部房间切换 Tab
   - 房间背景场景
   - 房间内成员分布

4. **main.js** - 房间切换逻辑：
   - 点击 Tab 切换房间
   - 不同房间显示不同成员
   - 房间切换动画

### Phase 2：成员状态增强

1. **data.js** - 扩展成员数据：
   - 当前所在房间
   - 当前行为（编程中/开会中/休息中）
   - 当前工作区/项目
   - 状态时长（多久没变化）

2. **styles.css** - 状态可视化增强：
   - 状态图标/标识
   - 行为动画（敲键盘、开会的动画）
   - 工作区 indicator

3. **main.js** - 状态展示：
   - 成员卡片显示更多信息
   - 实时更新时间
   - 状态历史

### Phase 3：UI 升级

1. **像素风增强**：
   - 更精致的办公室场景
   - 更多像素装饰（植物、地图、时钟等）
   - CRT 效果优化

2. **动效升级**：
   - 成员移动动画
   - 状态变化动画
   - 房间切换过渡

3. **响应式适配**：
   - 移动端基本可用
