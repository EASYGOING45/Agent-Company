## 🎯 任务目标

实现成员状态管理系统，支持：
- 四种状态：online / idle / busy / offline
- 每个成员有可见的状态文案（如"在线"、"摸鱼中"、"工作中"、"离线"）
- 成员卡片直接展示状态
- 当前用户可点击自己的卡片快速编辑状态
- 状态持久化到 localStorage

## 📦 涉及文件

- `src/data.js` - 成员状态数据模型、状态映射
- `src/main.js` - 状态切换逻辑、事件绑定
- `src/styles.css` - 状态样式、状态灯颜色
- `index.html` - 状态入口 UI
