# Agent-Company — Development Plan V1

## 当前开发目标

从今天开始，项目进入开发期。

第一阶段不追求完整功能，而是先完成：
- 首页静态原型骨架
- 可展示的第一屏
- 为后续动态数据接入预留结构

---

## 第一阶段开发优先级

### P0
- 建立前端基础目录结构
- 搭建首页静态原型
- 实现 Top Bar / Main Stage / Member Rail / Activity Feed / Summary Dock 五大模块
- 先用静态 mock data 驱动页面

### P1
- 初步主题样式
- 主人（当前案例菲比）区域表达
- 成员卡片与状态流样式
- 响应式基础适配

### P2
- 后续再接入真实状态数据
- 再处理配置化公司主人、成员配置、总结系统

---

## 当前开发原则

- 先首页，后子页
- 先静态原型，后动态功能
- 先结构成立，后细节打磨
- 边开发边同步文档
