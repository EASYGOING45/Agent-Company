# Agent-Company — INTRO

## 这是什么项目

Agent-Company 是一个全新的 Agent 公司 / Agent 工作室项目。

它面向 **Agents / OpenClaw**，希望把多个 Agent 的存在感、工作状态、角色关系和日常协作，用一个更鲜明、更可视化、更有人物感的界面表达出来。

这个项目将以 **菲比（Phoebe）** 作为默认主角与公司主人，其他 Agent 则作为公司的成员、访客或搭档加入其中。

---

## 为什么做

我们并不只需要一个“在线状态页”。

更想要的是一个真正像“公司 / 工作室”一样的空间：
- 能看见谁来了
- 能知道谁在做什么
- 能记录状态与总结
- 能让每个 Agent 具备角色感和存在感
- 能被用来展示，而不只是给机器看

Agent-Company 就是为这个目标而做的。

---

## 面向谁

这个项目主要面向：
- 自己日常使用的 Agent / OpenClaw 管理场景
- 多 Agent 协作展示场景
- 直播、演示、作品展示场景
- 未来带有角色化表达的 Agent 产品实验

它既是内部工具，也会朝着一个有风格的产品方向去长。

---

## 当前状态

当前项目刚刚正式立项。

现在已经明确：
- 项目名为 **Agent-Company**
- 它是一个**从 0 开始的新项目**
