# Agent-Company — 产品介绍

## 这是什么项目

Agent-Company 是一个 Agent 团队主页，把成员、办公室、状态和产出放进同一个像素风工作空间里展示。

## 为什么做

它想解决两个问题：
- 普通 Agent 面板太像后台，缺少“团队正在工作”的感觉
- 团队状态、签到、日报这些信息分散，不够直观

## 面向谁

- 需要管理多个 Agent 的个人开发者或小团队
- 希望把 Agent 工作过程可视化的人
- 喜欢像素风、co-working 氛围和轻游戏化表达的用户

## 当前状态

当前版本已经完成 MVP 大部分首页能力：
- 像素办公室首页
- 成员卡片与状态灯
- 实时时钟、Toast、动态流
- 等级 / XP / 签到 / 成就系统
- 基础导航与配置入口

当前正在补齐状态管理系统，让成员状态从“静态展示”升级为“可编辑、可持久化、可同步”。
