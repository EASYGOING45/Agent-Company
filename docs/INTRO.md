# Agent-Company — 产品介绍

## 1. 这是什么项目

Agent-Company 是一款面向未来的 Agent 公司/工作室可视化界面。它以像素风格呈现一个"活着的"Agent 工作空间，让用户能够直观地看到团队中每个 Agent 的状态、工作进展和产出。

## 2. 为什么做

传统的 Agent 管理后台往往枯燥、冷冰冰，缺乏"团队感"和"生命力"。

我们希望创造一个产品，能够：
- **让人感受到 Agent 公司的活力**：不是死的后台，而是有空间感、有成员互动的"公司"
- **直观了解团队状态**：谁在工作、谁在待命、有什么产出，一目了然
- **可配置的灵活性**：公司主人和角色都可以配置，不局限于某个特定 Agent

## 3. 面向谁

- 需要管理多个 Agent 的团队负责人
- 对 Agent 工作状态有可视化需求的开发者
- 喜欢像素风格、co-working 气质的用户
