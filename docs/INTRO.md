# Agent-Company — 产品介绍 v2.0

## 这是什么项目

Agent-Company 是一个**元宇宙风格的 Agents 可视化平台**。

用「多房间虚拟办公室」来表达 Agents 的存在感、状态、行为和当前工作区——你知道每个 Agent 在哪个房间、在做什么、进度如何。

## 为什么做

它想解决一个问题：

> **Agents 在干什么？看不清、不直观、缺少"团队感"**

传统 Agent 面板太像后台。Agent-Company 用虚拟办公室的方式，让 Agents "活"在一个空间里，像一个真正的团队在工作。

## 面向谁

- 管理多个 Agent 的开发者
- 需要直观看到 Agent 工作状态的场景
- 喜欢像素风、元宇宙、赛博朋克风格的用户
- 对 AI 团队协作可视化感兴趣的人

## 核心特性

### 多房间系统
- **主厅** - 核心办公区，成员常态工作
- **会议室** - 开会、讨论、头脑风暴
- **休息室** - 放松、交流、喝茶
- **游戏室** - 娱乐、互动、解压

### 实时状态
- 成员当前状态（在线/摸鱼/忙碌/离线）
- 当前行为（编程中/开会中/休息中）
- 所在房间
- 状态时长

### 像素风美学
- 复古像素 + 赛博朋克
- CRT 扫描线效果
- 霓虹光晕
- 精致动效，办公室"活"起来

### 角色化成员呈现
- 菲比使用本地头像资源，稳定可控
- 其他成员使用定制像素肖像，保留各自分工气质
- 焦点成员、房间点位、状态提示统一为一套像素光效语言
