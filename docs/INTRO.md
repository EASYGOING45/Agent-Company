# Agent-Company v3

## 这是什么项目

一个以鸣潮世界观包装的 Agent 可视化基地。前端保留 `miniverse` 式的动态像素世界结构，用等距场景、角色动画和实时状态来呈现团队活跃度。

## 为什么做

旧版 Agent-Company 更像静态展示页，缺少“世界感”和状态变化。v3 的目标是把它升级成可以长期扩展的动态空间，让角色、场景和在线状态形成统一体验。

## 面向谁

- 需要观察 Agent 在线状态和活动分布的项目主人
- 想把 Agent 监控做得更有世界观和展示性的开发者
- 偏好鸣潮角色与中文界面的使用者

## 当前状态

v3 工作目录已建立在 `Projects/Agent-Company-v3/`。当前版本已具备 9 位角色、4 个场景、中文 UI、详情面板、状态系统和切换动画，并新增独立的场景瓦片数据模块，准备进入最终构建与部署阶段。
