# Agent-Company — Asset Shortlist V1

## 1. 目标

这份文档用于明确：首页静态原型第一阶段，最少需要哪些素材。

目标不是一次找全，而是先满足 **首页静态原型可落地**。

---

## 2. 第一批必须素材

## 2.1 空间背景
- 像素办公室主背景（1 张）
- 可选：同场景近景 / 变体（1 张）

用途：
- 首页主舞台
- 确定空间基调

## 2.2 家具 / 场景元素
- 桌子 / 工位
- 椅子
- 植物
- 屏幕 / 墙面小物件
- 沙发或休息区元素

用途：
- 丰富舞台层次
- 区分工作区 / 休息区 / 主人区域

## 2.3 主人角色案例素材（菲比）
- 头像（1）
- 主视觉立绘 / 半身（1）
- 像素化角色占位或参考（1）

用途：
- 首页主人展示
- 成员卡片的 owner 案例

注意：
- 当前阶段可先用参考图 + 占位方案
- 后续再逐步转成更统一的项目资产

## 2.4 成员默认素材
- 默认成员头像占位（1~2）
- 默认访客头像占位（1~2）
- 在线 / 工作 / 待命 / 异常状态小图标

用途：
- 让成员列表先跑起来

## 2.5 UI 基础素材
- 像素按钮样式参考
- 小标签底板
- 状态点 / badge
- 小型卡片底板

用途：
- 首页成员带
- 状态流
- 总结层

---

## 3. 第一批可直接参考旧项目的内容

来自 StarOffice，可参考：
- office 背景构图
- desk / sofa / plants / posters / serverroom 的空间表达
- 像素家具氛围
- guest 角色在空间中出现的方式

但当前阶段建议：
- **参考结构，不直接照搬命名与角色语义**

---

## 4. 当前素材获取优先顺序

### 优先级 A
- 办公室背景
- 主人案例图（菲比）
- 成员默认占位

### 优先级 B
- 家具和装饰元素
- UI 小组件底板

### 优先级 C
- 动画素材
- 状态切换小动效
- 多主题可替换素材

---
